Accessing Elements

.indexOf()

Cuando quieras encontrar el PRIMER INDEX (de un valor dado dentro) de un array o string. const randomStuff = ['credit card', 'keys', 'receipt', 'gum', 'key', 'keys', 'plastic spoon']; Creamos una función que tenga como argumento un array const buscaIndex = array => array.indexOf('keys'); El método .indexOfbusca dentro de un array o un string si el valor dado existe, y si existe, devuelve el número INDEX en el que se encuentra. Si no existe, regresa '-1'. buscaIndex(randomStuff); //result 1 (index) Este metodo acepta un segundo parametro que indica desde que INDEX comienza a buscar el valor: const buscaIndex = array => array.indexOf('keys',2); buscaIndex(randomStuff); //result 5

Iterating Elements

.map() || .toUpperCase

¿Cómo agrego un caracter extra a un array de strings y convierto todas las letras a mayúsculas? const greetings = ['hello', 'hi', 'heya', 'oi', 'hey', 'yo']; Hago una función que itere sobre cada elemento, y que al mimso tiempo agregue el caracter const agrega = arr => arr.map(item => item.toUpperCase() + '!'); El metodo .map() itera sobre cada elemento para que lo uses a tu gusto. El metodo .toUpperCase() transforma un item string a MAYUSCULAS. agrega(greetings); //result ["HELLO!", "HI!", "HEYA!", "OI!", "HEY!", "YO!"]

.filter() | .includes()

¿Cómo comparar si dos arrays tienen los mismos elementos? : const arr1 = ['uno','dos','tres','cuatro','cinco']; const arr2 = ['uno','ocho','trece','cinco','diez','dos']; Declaramos la expresion de una función (ES6) que compare los dos arrays: const comparacion = (firstArray, secondArray) => firstArray.filter(item => secondArray.includes(item)); El método .filter itera sobre todos los elementos y devuelve un nuevo array con los elementos que hayan cumplido cierta condición.
El método .includes devuelte TRUE/FALSE si el array contiene el elemento especificado
comparacion(arr1,arr2); // result: ["uno", "dos", "cinco"]

.every()

¿Cómo saber si todos los elementos (en un array de objetos) tienen el mismo valor en una propiedad? const dinner = [{name: 'cheese', source: 'plant'}, {name: 'ketchup', source:'plant'}, {name: 'bun', source: 'plant'}, {name: 'twinkies', source:'plant'}]; Declaro la expresión de una función que acepte como argumento un array, para que dentro de la funsión se itere sobre cada elemento const mismoValor = (arr) => arr.every(item => item.source === 'plant'); El método .every() itera sobre cada elemento y devuelve TRUE/FALSE si la condición se cumple para TODOS los elementos. mismoValor(dinner); //result: true

Mutating Elements

.sort() | .reverse()

Ordena un array de mayor a menor: const years = [1970, 1999, 1951, 1982, 1963, 2011, 2018, 1922]; Declaramos la expresión de una función que tome como argumento un array, lo ordene y despues lo ponga al revés const ordena = arr => arr.sort().reverse(); El método .sort() analiza cada elemento del array y pone el mas chico primero y el grande al último. El método .reverse() toma al array que esta antes del '.' y lo regresa al revés. ordena(years); // result: [2018, 2011, 1999, 1982, 1970, 1963, 1951, 1922]
